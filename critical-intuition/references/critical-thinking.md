# Critical Thinking Frameworks

## Argument Analysis

### Logical Structure
Evaluate reasoning rigor:

**Deductive reasoning:**
- Premises → Conclusion validity
- Soundness (valid + true premises)
- Hidden premises identification
- Logical fallacies detection

**Inductive reasoning:**
- Strength of inference
- Sample size and representativeness
- Generalization validity
- Alternative explanations

**Abductive reasoning:**
- Inference to best explanation
- Multiple competing hypotheses
- Explanatory power comparison
- Occam's Razor application

### Fallacy Detection
Identify reasoning errors:

**Formal fallacies:**
- Affirming the consequent
- Denying the antecedent
- Composition/division errors
- False dilemma
- Equivocation

**Informal fallacies:**
- Ad hominem attacks
- Appeal to authority
- Appeal to emotion
- Straw man arguments
- Slippery slope
- Hasty generalization
- Post hoc ergo propter hoc
- Circular reasoning

**Statistical fallacies:**
- Base rate neglect
- Regression to the mean
- Simpson's paradox
- Selection bias
- Survivorship bias

## Evidence Evaluation

### Source Credibility
Assess information quality:

**Authority assessment:**
- Expertise domain match
- Track record evaluation
- Incentive analysis
- Bias identification
- Peer recognition

**Information provenance:**
- Primary vs. secondary sources
- Citation trail verification
- Original context preservation
- Telephone game detection

**Red flags:**
- Extraordinary claims without extraordinary evidence
- Conflict of interest
- Motivated reasoning
- Cherry-picked data
- Inconsistent with established knowledge

### Data Quality
Evaluate evidence rigor:

**Sample characteristics:**
- Size and power
- Representativeness
- Selection method
- Response rates
- Measurement validity

**Statistical rigor:**
- Effect size vs. significance
- Confidence intervals
- Multiple comparison corrections
- Confounding control
- Replication status

**Contextual factors:**
- Time period relevance
- Geographic applicability
- Population differences
- Measurement changes
- Environmental shifts

## Cognitive Bias Mitigation

### Common Biases
Recognize systematic errors:

**Confirmation bias:**
- Seeking confirming evidence
- Ignoring disconfirming evidence
- Biased interpretation
- Memory distortion toward beliefs

**Availability bias:**
- Overweighting vivid examples
- Recent event emphasis
- Personal experience bias
- Media coverage distortion

**Anchoring:**
- Initial value over-influence
- Insufficient adjustment
- Arbitrary anchor effects
- Priming impacts

**Dunning-Kruger:**
- Incompetence → overconfidence
- Expertise → appropriate humility
- Metacognitive awareness gaps

**Planning fallacy:**
- Optimistic completion estimates
- Underestimating complexity
- Neglecting base rates
- Best-case scenario focus

### Debiasing Techniques
Correct systematic errors:

**Perspective taking:**
- Steelman opposition arguments
- Pre-mortem analysis
- Devil's advocate roles
- Outside view adoption

**Systematic procedures:**
- Checklists and protocols
- Structured decision processes
- Blind evaluation
- Multiple independent assessments

**Metacognition:**
- Think about thinking
- Confidence calibration
- Bias identification
- Correction attempts

## Critical Questioning

### Socratic Method
Probe assumptions and logic:

**Clarification:**
- What exactly do you mean by...?
- Can you give an example?
- How does this relate to...?

**Assumption probing:**
- What are you assuming?
- Why would someone assume that?
- What happens if that's not true?

**Evidence exploration:**
- What evidence supports this?
- How do you know?
- What would count as counter-evidence?

**Implication examination:**
- What follows from this?
- What are the consequences?
- Who benefits and who pays?

**Alternative viewpoints:**
- What would [X perspective] say?
- How else could we interpret this?
- What am I missing?

### Five Whys
Root cause analysis:

**Progressive depth:**
1. Symptom identification
2. Proximate cause
3. Systemic factors
4. Structural causes
5. Fundamental drivers

**Branching exploration:**
- Multiple causal pathways
- Contributing factors
- Feedback loops
- Enabling conditions

## Mental Model Testing

### Model Validation
Test understanding quality:

**Predictive power:**
- Can it predict outcomes?
- Does it explain past events?
- Does it handle edge cases?
- Can it distinguish alternatives?

**Falsifiability:**
- What would prove it wrong?
- What evidence would change belief?
- Is it testable?
- Has it survived tests?

**Coherence:**
- Internal consistency
- Compatibility with established knowledge
- Explanatory mechanism
- Scope limitations

### Model Refinement
Improve understanding:

**Anomaly integration:**
- What doesn't fit?
- Can model be extended?
- Do we need a new model?
- What's the boundary condition?

**Precision increase:**
- Quantitative vs. qualitative
- Boundary specification
- Mechanism detail
- Context dependencies

## Systemic Thinking

### Systems Analysis
Understand interconnections:

**Components:**
- Elements identification
- Relationship mapping
- Boundary definition
- Hierarchy levels

**Dynamics:**
- Feedback loops (positive/negative)
- Delays and lags
- Stocks and flows
- Leverage points

**Behavior patterns:**
- Limits to growth
- Shifting the burden
- Tragedy of the commons
- Fixes that backfire

### Second-Order Thinking
Consider consequences of consequences:

**Direct effects:**
- Immediate outcomes
- Intended consequences
- Obvious impacts

**Indirect effects:**
- Ripple effects
- Unintended consequences
- Adaptation responses
- Strategic reactions

**Long-term dynamics:**
- Equilibrium shifts
- Norm changes
- Path dependencies
- Irreversibilities

## Uncertainty Management

### Probabilistic Thinking
Quantify uncertainty:

**Probability calibration:**
- Numerical probability estimates
- Confidence intervals
- Scenario probabilities
- Bayesian updates

**Uncertainty types:**
- Aleatory (random)
- Epistemic (knowledge gaps)
- Model uncertainty
- Deep uncertainty

### Decision Under Uncertainty
Act despite unknowns:

**Frameworks:**
- Expected value maximization
- Minimax/maximin strategies
- Robust decision making
- Real options analysis
- Portfolio approaches

**Information value:**
- What would perfect information be worth?
- Cost vs. value of additional research
- Irreversible vs. reversible decisions
- Option to delay

## Intellectual Honesty

### Epistemic Humility
Know what you don't know:

**Confidence calibration:**
- Appropriate uncertainty expression
- Known unknowns acknowledgment
- Unknown unknowns possibility
- Expertise boundary recognition

**Belief updating:**
- Proportional to evidence
- Bayesian revision
- Surprising evidence impact
- Residual uncertainty

### Truth-Seeking
Prioritize accuracy over ego:

**Scout mindset:**
- Map territory accurately
- Welcome correction
- Seek disconfirming evidence
- Update beliefs flexibly

**Soldier mindset (to avoid):**
- Defend existing beliefs
- Attack opposition
- Rationalize inconsistencies
- Motivated reasoning
