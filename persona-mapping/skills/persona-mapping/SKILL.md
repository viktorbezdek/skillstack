# Persona Mapping

Map stakeholders by influence, interest, and responsibility.

## Power-Interest Matrix

```
    HIGH INTEREST
         |
  KEEP   |   MANAGE
SATISFIED|   CLOSELY
         |
LOW -----+----- HIGH POWER
         |
 MONITOR |    KEEP
         |   INFORMED
    LOW INTEREST
```

## RACI Matrix

| Role | Definition | Per Task |
|------|------------|----------|
| **R**esponsible | Does work | 1+ |
| **A**ccountable | Decides | 1 |
| **C**onsulted | Input (2-way) | 0+ |
| **I**nformed | Updates (1-way) | 0+ |

## Stakeholder Template

```markdown
## Stakeholder: [Name]

- **Influence**: High/Med/Low
- **Interest**: High/Med/Low
- **Attitude**: Supportive/Neutral/Resistant

### Engagement Strategy
- **Frequency**: [cadence]
- **Channel**: [method]
- **Key messages**: [emphasis]
```

## Salience Model

| Attributes | Type | Priority |
|------------|------|----------|
| Power + Legitimacy + Urgency | Definitive | Highest |
| Power + Legitimacy | Dominant | High |
| Single attribute | Latent | Low |
