# Creative Problem Solving

> **v1.0.15** | Strategic Thinking | 16 iterations

Generate breakthrough solutions through lateral thinking, first principles reasoning, game theory, and strategic reframing.

## What Problem Does This Solve

This skill enables sophisticated creative and strategic thinking to help users find optimal, unconventional, and high-leverage solutions to complex problems.

## When to Use This Skill

Generate new ideas and breakthrough solutions through brainstorming, lateral thinking, SCAMPER, first principles reasoning, game theory, strategic reframing, and innovative problem-solving.

## When NOT to Use This Skill

- analyzing or critiquing existing ideas, bias detection, or pattern recognition in evidence -- use [critical-intuition](../critical-intuition/) instead
