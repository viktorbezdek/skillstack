# Creative Problem-Solving

> Generate breakthrough solutions through lateral thinking, first principles reasoning, game theory, and strategic reframing.

## Overview

When conventional approaches fail or when a problem seems stuck, what is needed is not more effort in the same direction but a fundamentally different perspective. This skill provides structured creativity -- systematic methods for breaking mental blocks, challenging assumptions, reframing problems, and discovering high-leverage solutions that others miss.

The Creative Problem-Solving skill combines five complementary thinking methodologies: lateral thinking techniques (random entry, provocation, assumption escape), strategic frameworks (game theory, first principles, systems thinking), reframing methods (abstraction shifts, perspective rotation, constraint manipulation), ideation techniques (SCAMPER, morphological analysis, TRIZ), and decision frameworks (multi-criteria analysis, probabilistic assessment, bias mitigation). It follows a five-step process: understand deeply, reframe strategically, generate broadly, analyze rigorously, and recommend optimally.

Within the SkillStack collection, this skill pairs powerfully with Critical Intuition (detecting hidden patterns that inform creative solutions), Code Review (finding unconventional approaches to architectural problems), and Content Modelling (innovative content architecture design).

## Key Features

- **Lateral thinking toolkit**: Random entry, provocation, and assumption escape techniques to break past mental blocks
- **First principles reasoning**: Decompose problems to fundamental truths, rebuild solutions without conventional assumptions
- **Game theory analysis**: Map incentives, find Nash equilibria, identify positive-sum opportunities, and detect misaligned incentives
- **Strategic reframing**: View problems through abstraction shifts, perspective rotation, inversion, and constraint manipulation
- **Systems thinking**: Identify feedback loops, leverage points, second-order effects, and time delays
- **Probabilistic reasoning**: Expected value calculations, Bayesian updating, base rate analysis, and tail risk assessment
- **Structured ideation**: SCAMPER, morphological analysis, and cross-domain transfer for systematic idea generation
- **Decision optimization**: Multi-criteria analysis, reversibility testing, and real options valuation for sound recommendations

## Usage Examples

**Find unconventional competitive strategies:**
```
How can our 10-person startup compete with established companies
that have 100x our resources? Think outside the box.
```
Expected output: A strategic analysis that reframes the problem from "how to match resources" to "how to win differently," applying game theory to identify where competitors are locked into commitments, first principles thinking about what customers truly value, and specific asymmetric strategies (niche domination, speed, relationship depth) with probability-weighted expected outcomes.

**Break through a technical constraint:**
```
Our system processes 10M events/day but we need to handle 100M.
The database is the bottleneck and we can't change it.
Use first principles to find creative solutions.
```
Expected output: Challenge of the "can't change it" assumption (physical vs. political constraint?), first principles decomposition of what the database actually does, lateral thinking options (pre-aggregation, event sourcing, CQRS, edge processing), systems thinking analysis of feedback loops, and a recommended approach with implementation steps.

**Resolve an organizational deadlock:**
```
Two teams can't agree on a shared API design. Each wants their
preferred approach. Apply game theory to find a resolution.
```
Expected output: Incentive mapping for each team, identification of the game type (coordination vs. prisoner's dilemma), Nash equilibrium analysis, positive-sum reframing opportunities, and specific proposals that align individual and collective incentives.

**Generate innovative product ideas:**
```
We have a successful B2B analytics product. Brainstorm creative
ways to expand into adjacent markets using SCAMPER and cross-domain
transfer.
```
Expected output: Systematic SCAMPER analysis (Substitute, Combine, Adapt, Modify, Put to other use, Eliminate, Reverse) applied to the product's core capabilities, cross-domain analogies from consumer tech and other industries, and a prioritized list of opportunities ranked by expected value and feasibility.

**Make a high-stakes decision under uncertainty:**
```
We need to choose between building a feature ourselves (6 months,
$500K) or acquiring a startup that has it ($2M, immediate).
Help me think through this strategically.
```
Expected output: Expected value analysis for both options with probability ranges, real options analysis (what does each choice enable or foreclose?), reversibility assessment, scenario analysis for best/worst/likely outcomes, identified biases to watch for, and a clear recommendation with decision criteria for when to choose the alternative.

## Quick Start

1. **Define the real problem** -- Go beyond the stated problem to the underlying goal and constraints
2. **Challenge assumptions** -- List all implicit assumptions and identify which can be questioned
3. **Choose your thinking tools** -- Select from lateral thinking, game theory, first principles, or systems thinking based on the problem type
4. **Load the right reference** -- Read the relevant reference file for detailed techniques
5. **Generate divergently** -- Use structured methods to explore a wide solution space
6. **Analyze convergently** -- Apply decision frameworks to evaluate and rank solutions
7. **Recommend with clarity** -- Present the primary recommendation with reasoning, risks, and concrete next steps
