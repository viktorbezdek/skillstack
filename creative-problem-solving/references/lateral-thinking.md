# Lateral Thinking Techniques

## Random Entry
Introduce random stimulus to break fixed patterns.
- Pick random word/object → Force connections to problem
- "How could [random concept] solve this?"
- Use dictionary, images, or environmental prompts

## Provocation
Make deliberately unreasonable statements to spark new directions.
- PO (Provocative Operation): "PO: customers pay us to NOT use our product"
- Reversal: "What if we did the opposite?"
- Exaggeration: "What if this took 100x longer/faster?"
- Escape: "What if [core assumption] wasn't true?"

## Challenge Assumptions
List and systematically question every assumption.
- "Why must it be this way?"
- "What if we couldn't use [standard method]?"
- "Who says this is the problem we should solve?"

## Concept Fan
Branch thinking from abstract to concrete.
- Why? → Purpose → Broader concepts → Alternative approaches
- Fixed point → Broader purpose → Different fixed points
- Example: "Transport" → "Get from A to B" → "Eliminate need to travel"

## Movement
Use ideas as stepping stones, not endpoints.
- Extract value from "wrong" ideas
- Ask "What's interesting about this?" not "What's wrong?"
- Build forward: "Yes, and..." over "No, but..."
